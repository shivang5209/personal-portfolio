# Personal Portfolio Website

One-page personal portfolio website with a blue glass theme, animated sections, project cards, contact CTA, and hero portrait.

## Run Locally

Open `index.html` directly, or run:

```bat
start-portfolio.bat
```

Then visit:

```text
http://127.0.0.1:4173/
```

## Edit Content

Most portfolio content is in `src/main.js` inside the `portfolio` object.

## GitHub Pages CI/CD

This project includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.

When pushed to the `main` branch, GitHub Actions deploys the static site to GitHub Pages.
